1.	Objetivo
O objetivo deste documento é detalhar os campos e regras das interfaces relacionadas ao caso de uso “Mantém Parâmetro de Mensagem Digital”.
2.	Detalhamento da apresentação
O acesso a esta funcionalidade se dá através do menu: Parâmetros >> Mensagens Digitais.
2.1.	Interface Consulta Parâmetro Mensagem Digital
2.1.1.	Campos
	Estímulo
-	Descrição: Permite selecionar o estímulo para a consulta da mensagem digital;
-	Formato: Alfanumérico;
-	Nome do campo na interface: “Estímulo”;
-	Máscara: Não se aplica;
-	Tamanho: 30;
-	Valores possíveis: Não se aplica;
-	Regras de interface:
-	Campo editável;
-	Campo não obrigatório;
-	Campo do tipo caixa de seleção.
Dados do rodapé:
	Usuário autenticado na aplicação
-	Descrição: Apresenta a identificação do usuário que está autenticado na aplicação;
-	Nome do campo na interface: “Usuário”;
-	Tamanho: 10;
-	Valores possíveis: Conforme dados obtidos do SIPCS;
-	Campo não editável.
2.1.2.	Opções
	Desfazer
-	Descrição: Quando acionada, limpa os campos de consulta.
	Consultar
-	Descrição: Quando acionada, apresenta a interface “2.2 - Interface Apresenta Resultado da Consulta Parâmetro Mensagem Digital”.
	Incluir Novo Registro
-	Descrição: Quando acionada, apresenta a interface “2.44 - Interface Inclui Parâmetro Mensagem Digital”.
2.1.3.	Regras de Apresentação
-	Não se aplica.
2.2.	Interface Apresenta Resultado da Consulta Parâmetro Mensagem Digital
2.2.1.	Campos
Os campos dos filtros de consulta e do rodapé são os mesmos descritos em 2.1.1.
Os demais campos da interface estão abaixo:
Resultado da Consulta:
	Quantidade de registros encontrados
-	Descrição: Apresenta o número de registros encontrados para a pesquisa;
-	Nome do campo na interface: “XXX Registro(s) encontrado(s)”, onde XXX representa o número de registros encontrados;
-	Tamanho: 03;
-	Descrição: Apresenta o estímulo cadastrado;
-	Campo não editável;
-	Campo permite acesso à interface “2.3 - Interface Detalha Parâmetro de Mensagem Digital”.
	Título da Mensagem
-	Descrição: Apresenta o título da mensagem digital cadastrada;
-	Nome do campo na interface: “Título da Mensagem”;
-	Tamanho: 50;
	Situação
-	Descrição: Apresenta a situação da mensagem cadastrada;
-	Nome do campo na interface: “Situação”;
Paginação:
	Paginação (Página Anterior)
-	Descrição: Apresenta a página anterior do resultado da consulta;
-	Nome do campo na interface: “<<”;
-	Tamanho: Não se aplica;
-	O campo permite acesso à interface anterior do resultado da consulta.
	Número da Página
-	Descrição: Apresenta os números das páginas;
-	Nome do campo na interface: “XX”, onde XX representa o número da página do resultado da consulta;
-	Tamanho: 02;
	Paginação (Próxima Página)
-	Descrição: Apresenta a próxima página do resultado da consulta;
-	Nome do campo na interface: “>>”;
-	O campo permite acesso à próxima página do resultado da consulta.
2.2.2.	Opções
	 As opções “Consultar”, “Desfazer” e “Incluir Novo Registro” são as mesmas descritas em 2.1.2.
2.2.3.	Regras de apresentação
-	A lista de resultados é apresentada ordenada de forma alfabética pelo campo “Estímulo”.
-	O sistema deve paginar a lista quando a pesquisa retornar como resultado mais de 5 registros.
-	O campo “Paginação (Página Anterior)” é apresentado somente a partir da segunda página de resultados da consulta.
-	O campo “Paginação (Próxima Página)” é apresentado caso haja mais de uma página para o resultado da pesquisa.
2.3.	Interface Detalha Parâmetro de Mensagem Digital
2.3.1.	Campos
O campo do rodapé é o mesmo descrito em 2.1.1.
-	Nome do campo na interface: “Estímulo*”;
	Quantidade de dias de atraso (De)
-	Descrição: Apresenta a quantidade de dias de atraso de (igual ou maior);
-	Formato: Numérico;
-	Nome do campo na interface: “De (igual ou maior)*”;
-	Máscara: 999;
-	Tamanho: 3;
	Quantidade de dias de atraso (Até)
-	Descrição: Apresenta a quantidade de dias de atraso até (menor ou igual);
-	Nome do campo na interface: “Até (menor ou igual)*;
-	Nome do campo na interface: “Situação*”;
	Data Situação
-	Descrição: Apresenta a data da situação da mensagem cadastrada;
-	Formato: Data;
-	Nome do campo na interface: “Data Situação*”;
-	Máscara: DD/MM/AAAA;
-	Valores possíveis: Conforme dados obtidos no SIPCS;
-	Nome do campo na interface: “Título da Mensagem*”;
	Mensagem
-	Descrição: Apresenta o texto da mensagem digital cadastrada;
-	Nome do campo na interface: “Texto da Mensagem*”;
-	Tamanho: 245;
2.3.2.	Opções
	Voltar
	A opção “Desfazer” é a mesma descrita em 2.1.2.
	Alterar
-	Descrição: Quando acionada, apresenta a interface “2.5 - Interface Altera Parâmetro Mensagem Digital”.
	Excluir
-	Descrição: Quando acionada, apresenta a interface “2.6 - Interface Exclui Parâmetro Mensagem Digital”.
2.3.3.	Regras de apresentação
	A opção “Desfazer” deve ser apresentada desabilitada.
2.4.	Interface Inclui Parâmetro Mensagem Digital
2.4.1.	Campos
-	Descrição: Permite informar a quantidade de dias de atraso de (igual ou maior);
-	Campo obrigatório;
-	Campo do tipo caixa de texto.
-	Descrição: Permite informar a quantidade de dias de atraso até (menor ou igual);
-	Nome do campo na interface: “Até (menor ou igual)*”;
-	Descrição: Permite informar o título da mensagem digital;
-	Descrição: Permite informar o texto da mensagem digital;
2.4.2.	Opções
-	Descrição: Quando acionada, retorna à interface anterior.
	 Desfazer
-	Descrição: Quando acionada, limpa os campos de inclusão.
	Incluir
-	Descrição: Quando acionada, registra a inclusão de parâmetro de mensagem digital.
2.4.3.	Regras de apresentação
	A opção “Excluir” deve ser apresentada desabilitada.
2.5.	Interface Altera Parâmetro Mensagem Digital
2.5.1.	Campos
-	Descrição: Apresenta o estímulo da mensagem digital;
-	Descrição: Permite selecionar a situação da mensagem digital.
	Data da Situação
-	Descrição: Apresenta a data da situação da mensagem digital;
-	Tamanho:10;
-	Descrição: Permite alterar o título da mensagem digital;
-	Descrição: Permite alterar o texto da mensagem digital;
2.5.2.	Opções
-	Descrição: Quando acionada, limpa os campos.
	Salvar
-	Descrição: Quando acionada, registra a alteração das informações de mensagem digital.
2.5.3.	Regras de apresentação
	O campo “Situação” será permitido apenas selecionar a opção “Ativo”, caso esteja “Inativo”.
2.6.	Interface Exclui Parâmetro Mensagem Digital
2.6.1.	Campos
2.6.2.	Opções
-	Descrição: Quando acionada, registra a exclusão das informações de mensagem digital.
2.6.3.	Regras de apresentação
	A opção “Alterar” deve ser apresentada desabilitada.
	O campo “Situação” será permitido apenas selecionar a opção “Inativo”, caso esteja “Ativo”.
